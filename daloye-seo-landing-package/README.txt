# 大老爺娛樂城 SEO 首頁包

## 結構
- index.html：首頁
- article/：6 篇文章頁
- img/：圖片預留位置與 SVG 佔位圖

## 你之後要替換的圖片檔名
- /img/logo.png
- /img/banner-01.jpg
- /img/banner-02.jpg
- /img/banner-03.jpg
- /img/provider-01.jpg
- /img/provider-02.jpg
- /img/provider-03.jpg
- /img/provider-logo-01.png
- /img/provider-logo-02.png
- /img/provider-logo-03.png
- /img/promo-01.jpg ~ /img/promo-06.jpg

目前資料夾內放的是同名的 SVG 佔位版本（例如 banner-01.jpg.svg），
如果正式圖片尚未上傳，頁面會自動 fallback 顯示佔位圖。
